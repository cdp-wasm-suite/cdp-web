# Demo source sounds

Small example sounds used by the audio-file recipes (File ▸ Recipes). They are
CDP's own demonstration material, taken from the CDP8 documentation bundle:

| File | Source in the CDP docs | Character |
| --- | --- | --- |
| `marimba.wav` | `docs/html/txsnds/marimba.wav` | short pitched percussion |
| `horn.wav` | `docs/demo/cdpexamples/horn.wav` | sustained instrument tone |
| `speech.wav` | `docs/htmltuts/tut12step/count.wav` | spoken counting voice |
| `frog.wav` | `docs/demo/sdbats/wavsources/frogcdt.wav` | textural / animal sound |

All are mono, 44.1 kHz. They ship with the Composers Desktop Project documentation
and are reused here under the same terms as the rest of the CDP material in this repo.
