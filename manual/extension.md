The **Composers Desktop Extension** embeds cdp-web inside Ableton, 
so you can transform a clip without leaving your session.

## What's different from the browser

- **Your selection arrives as a Source.** When the extension opens, the audio
  you had selected in the host is dropped in as a Source node automatically, so
  you can start patching straight away.
- **Output nodes get an "Apply" button.** In place of the browser's *Save* button, 
  each [Output](interface.md#the-output-node-rendering-and-the-stale-grey-out)
  shows an **Apply → _host_** button. It renders the chain to the Abelton sesssion or 
  arrangement, and closes the dialog.
- Everything else — sampler keyboard, minimap, log, auto-render — behaves as in
  the browser.
